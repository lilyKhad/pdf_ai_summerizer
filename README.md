## ✨ Features

- **Authentication** — Sign up and log in securely via Supabase Auth
- **PDF Upload** — Upload documents directly to Supabase Storage
- **AI Summarization** — Summaries generated by Groq AI (Llama 3.3 70B), structured with an overview, key points, and conclusion
- **AI Flashcards** — Automatically generate study flashcards from any document, with a 3D flip animation to reveal answers
- **Scanned PDF support** — Falls back to vision model (`llama-4-scout`) for image-based PDFs with no text layer
- **Local flashcard caching** — Generated flashcards are saved with Hive so they load instantly on revisit
- **Document history** — View all previously uploaded and summarized documents

## 🏗️ Architecture

This project follows **Clean Architecture** with a feature-first folder structure:

```
lib/
├── core/
│   ├── injection/        # Riverpod providers (dependency injection)
│   ├── router/           # App navigation & theme (AppRouter, AppTheme)
│   └── utils/            # PDF text extraction utilities
│
├── features/
│   ├── auth/
│   │   ├── data/         # Models, datasources, repository impl
│   │   ├── domain/       # Entities, repository interface
│   │   └── presentation/ # Screens, providers
│   │
│   ├── pdf/
│   │   ├── data/         # Models, Groq & Supabase datasources, repository impl
│   │   ├── domain/       # Entities, repository interface
│   │   └── presentation/ # Home & summary screens, providers
│   │
│   └── flashcards/
│       ├── data/         # FlashcardModel, Groq datasource, Hive local cache, repository impl
│       ├── domain/       # FlashcardEntity, repository interface
│       └── presentation/ # Flashcard screen, FlashcardNotifier
│
├── .env                  # API keys (not committed)
└── run.bat               # Windows dev launch script
```

## 🚀 Getting Started

### Prerequisites

- Flutter SDK
- A [Supabase](https://supabase.com) project with:
  - Auth enabled
  - A storage bucket for PDFs
- A [Groq](https://console.groq.com) API key

### Setup

1. **Clone the repo**
   ```bash
   git clone https://github.com/your-username/pdf-summarizer.git
   cd pdf-summarizer
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Create your `.env` file** at the root of the project:
   ```env
   SUPABASE_URL=your_supabase_url
   SUPABASE_ANON_KEY=your_supabase_anon_key
   GROQ_API_KEY=your_groq_api_key
   ```

4. **Run the app**
   ```bash
   flutter run
   ```
   Or on Windows, use the provided script:
   ```bash
   run.bat
   ```

---

## 📁 Environment Variables

| Variable | Description |
|---|---|
| `SUPABASE_URL` | Your Supabase project URL |
| `SUPABASE_ANON_KEY` | Your Supabase anonymous public key |
| `GROQ_API_KEY` | Your Groq API key |

> ⚠️ Never commit your `.env` file. Make sure it's listed in `.gitignore`.
